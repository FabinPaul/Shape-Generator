# Shape-Generator
A simple android app to generate shapes

* Implemented in MVVM using Hilt
* Uses Custom Drawable to draw Shapes like Circle, Square, and Rectangle on an ImageView
* Validated user input dimension of the shapes against the size of the ImageView
